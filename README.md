# NoteHub

A Vite + React + TypeScript application for searching, creating, and deleting
notes through the NoteHub API.

## Setup

1. Install dependencies with `npm install`.
2. Copy `.env.example` to `.env` and add your NoteHub access token.
3. Start the development server with `npm run dev`.

The token is available from the
[NoteHub API documentation](https://notehub-public.goit.study/api/docs).
